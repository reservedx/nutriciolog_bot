# Telegram-бот нутрициолог

MVP Telegram-бота, который принимает фото еды, отправляет его в OpenAI для vision-анализа и возвращает ориентировочную оценку КБЖУ.

## Что умеет

- принимает фото блюда в Telegram
- определяет блюдо и примерный состав
- оценивает калории, белки, жиры и углеводы
- показывает допущения и короткий совет

## Быстрый старт

1. Установить зависимости:

```bash
npm install
```

2. Создать `.env` по примеру `.env.example`.

3. Указать:

```env
TELEGRAM_BOT_TOKEN=...
OPENAI_API_KEY=...
OPENAI_MODEL=gpt-4.1-mini
PORT=3000
```

4. Запустить:

```bash
npm run dev
```

## Как использовать

- отправить `/start`
- прислать фото еды
- получить оценку КБЖУ
